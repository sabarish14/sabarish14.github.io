Description

This project is developed as a course project for Data mining course.This is a backend application which predicts the sentiment of a tweet as either positive,negative or neutral.

The project was done with the following goals in mind:

  • To develop machine learning model to predict the sentiment of a tweet
  • The predictive model should be immune to skewness of the data and have a high precision and recall

    • Reflections

    After analysis of the training dataset, a number of features were identified and they are,

  • Hash Tags: Certain hashtags seemed to contribute towards the sentiment of the tweet as they are very frequent among a class of sentiment.So these hashtags were identified and were used as one of the features.Eg: hashtags like #tcot,#tlot seemed to occur frequently in the negative class of the obama dataset, hence these hashtags were identified
  • Unigram and bigrams: Unigram along with bigrams of each classifier was used to train the classifier . Both of these were also used as a feature
  • Unigram polarity calculation: Polarity values were calculated for each unigram corresponding to each class­positive,negative and neutral.
  • Bigram pointwise mutual information values: Bigrams are identified from each tuple and pointwise mutual information value is calculated.
  • Sentiment of the neighbouring words : The neighbouring words surrounding the words denoting obama and romney obtained in the polarity feature are used here also. The difference is that only the opinion words in these neighbouring words present in the positive and negative opinion words list prepared by Professor Bing Liu are considered. And if a word is preceded by “not” the sentiment is reversed. Now the sentiment of these words either +1 or ­1 are added and normalized by the number of opinion words.This value is also used as a feature
