Description

This project is developed as a course project for Big data Analytics course.This is a web application which has the following features:

  • Predicting flight delays when the itinerary is entered
  • Visualization of the busiest Airports across United States
  • Calendar visualization that shows the delays in a month

    • The project was done with the following goals in mind:

  • To develop machine learning models for big data (greater than 4 GB)
  • To develop a full stack web application that can use the prediction results of the machine learning model.
  • To develop visualizations by analyzing the big data

    • Reflections

    I have developed machine learning models for smaller datasets in my earlier courses. This project gave me more realistic experience of working with larger dataset. I set up a Hadoop cluster on an EC2 instance and developed the backend application on the cluster. The datab was stored in MongoDB. The features that were used are Departure date,Departure time,Origin, Destination,Airlines,Weather prediction. The machine learning model was built using Mahout library. The results of the prediction were stored in MongoDB. The front-end of the application was built using Node and Angular JS. The front-end had a form where users enters the itinerary details. Then the prediction result of the itinerary is fetched from MongoDB and displayed.

    Things I learned:

    • Hadoop framework is awesome. It handles so many nitty-gritty things.
    • Converting a sequential operation to a parallelized one was quite challenging
    • Analyzing a big dataset was very difficult.
